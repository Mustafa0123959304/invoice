
# تعليمات تشغيل برنامج الفواتير (Desktop)

هذا البرنامج مصمم ليعمل كبرنامج ديسكتوب على نظام التشغيل الخاص بك باستخدام لغة Python.

### 1. المتطلبات:
يجب أن يكون لديك **Python 3.x** مثبتاً على جهازك.

### 2. تثبيت المكتبات اللازمة:
افتح نافذة الأوامر (Terminal/CMD) وقم بتنفيذ الأمر التالي:
```bash
pip install fpdf2 arabic-reshaper python-bidi Pillow
```

### 3. تشغيل البرنامج:
قم بتشغيل الملف باستخدام الأمر:
```bash
python main.py
```

### 4. ملاحظة هامة حول الخطوط العربية:
للحصول على أفضل نتيجة في ملف الـ PDF، يفضل تحميل خط عربي بصيغة `.ttf` (مثل خط Amiri) ووضعه في نفس المجلد وتعديل السطر الخاص بـ `add_font` في الكود ليقرأ هذا الملف. الكود الحالي يستخدم خطوط النظام الافتراضية للتوضيح.

### المميزات:
- واجهة رسومية بسيطة وسهلة.
- دعم رفع شعار المؤسسة.
- حسابات تلقائية للضريبة والإجمالي.
- تصدير ملفات PDF جاهزة للطباعة.
